About Wavefont Optimizer

Wavefont Optimizer is a developer tool designed to help you visualize and optimize web font loading strategies.

Features

  • Test different loading strategies (Block, Swap, Fallback)
  • Visualize loading timelines and metrics (FOIT, FOUT, CLS)
  • Interactive playground for variable fonts
  • Export optimized CSS snippets
